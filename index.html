jQuery.verticalScroll.js

Enable vertical scroll on htmlElement for Android.

jquery.verticalScroll.js achieve vertical scrollof a vertical html element in Android devices, by using the transform3d of css3.
This plugin have been tested and approved for use only in the webkit-browser.
If there is a bug, enjoy hacking.

Lazy Scroll

$("#scroll2").verticalScroll({
  buffer:40,
  duration:50,
  height:400,
  response:'lazy',
});
